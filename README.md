# Raycast Scripts for macOS

[![GitHub](https://img.shields.io/github/license/hoishing/raycast-scripts)](https://opensource.org/licenses/MIT)

This repo contain scripts for [Raycast][raycast], with the following features:

- quickly open folders `open-*.sh`
- open current Finder folder in VSCode
- open [espanso][espanso] config folder in VSCode
- open current Finder folder in Terminal

## Installation

Put the scrips in Raycast's Script Command folder, default `~/raycast/scripts`
